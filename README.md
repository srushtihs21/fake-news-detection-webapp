# 📰 Fake News Detection Web App

A machine learning web application that detects whether news is **REAL or FAKE** using Natural Language Processing (NLP) and Flask.

---

## 🚀 Features
- 🧠 Machine learning model for prediction
- 🎨 Clean and modern UI
- ⚡ Instant results
- 📱 Responsive design

---

## 🛠 Tech Stack
- Python
- Flask
- HTML, CSS
- Scikit-learn
- NLP (TF-IDF)
